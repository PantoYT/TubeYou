📦 TubeYou – Local Apache Setup (XAMPP)
1. Enable .htaccess support

Make sure your project contains a .htaccess file in the public directory:

Options -MultiViews
RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^ index.php [L]
2. Enable Apache rewrite module

Open:

C:\xampp\apache\conf\httpd.conf

Find and uncomment:

LoadModule rewrite_module modules/mod_rewrite.so
3. Configure Virtual Host

Open:

C:\xampp\apache\conf\extra\httpd-vhosts.conf

Add:

<VirtualHost *:80>
    ServerName tubeyou.local
    DocumentRoot "PATH_TO_PROJECT\public"

    <Directory "PATH_TO_PROJECT\public">
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>

Replace:

PATH_TO_PROJECT

with your actual project path (e.g. E:\Pliki\Projects\TubeYou\public)

4. Local domain (hosts file)

Edit (as Administrator):

C:\Windows\System32\drivers\etc\hosts

Add:

127.0.0.1 tubeyou.local
5. Restart Apache

Restart Apache from XAMPP Control Panel.

6. Access the project

Open in browser:

http://tubeyou.local
🌐 Access from other devices (Wi-Fi network)

If you want to access the project from other devices in the same network:

Option A — Router / DNS (recommended)

Use your router’s DNS settings (e.g. “Static DNS”, “Local DNS”, or similar) and add:

tubeyou.local → YOUR_LOCAL_IP

Example:

tubeyou.local → 192.168.33.6

Make sure:

Devices use the router as DNS (default via DHCP)
Apache is accessible on the network
Firewall allows incoming connections on port 80

Then access:

http://tubeyou.local

from any device in the Wi-Fi network.

Option B — Manual hosts file (per device)

On each device, map the domain manually:

YOUR_LOCAL_IP tubeyou.local

Example:

192.168.33.6 tubeyou.local

Note:

Works on PCs easily
Mobile devices may require additional tools or root access
⚠️ Notes
tubeyou.local works locally via hosts or DNS
VirtualHost does not resolve domain names — it only handles incoming requests
For network-wide access, DNS (router or Pi-hole) is required
Make sure Apache is reachable from other devices (firewall + network)
